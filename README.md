## Advantage 框架的优势
* 1.文件少，代码简洁
* 2.手动导入工具类
* 3.具备较高自定义性


## Requirements 要求
* iOS 7+
* Xcode 8+


## Usage 使用方法
### 第一步 引入头文件
PNChartTools
### 第二步 简单调用
```
[OrderDic order:dic]
```
